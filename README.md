# 🛒 E-Commerce Backend (MERN + Cloudinary)

A scalable and production-ready e-commerce backend built using Node.js, Express, MongoDB, and Cloudinary.

---

## 🚀 Features

- 🔐 Authentication & Authorization (JWT)
- 👤 User Management (Admin / Customer)
- 📦 Product Management
- 🖼 Image Upload (Cloudinary)
- ❌ Delete Single Image (Cloudinary sync)
- ⭐ Set Primary Product Image
- 🧾 Order Management
- 🛒 Cart System
- 🔍 Product Search (Text Index)
- 📊 Pagination, Filtering & Sorting
- ⚡ Scalable Modular Architecture

---

## 🏗 Tech Stack

- **Backend:** Node.js, Express.js
- **Database:** MongoDB (Mongoose)
- **Authentication:** JWT
- **File Storage:** Cloudinary
- **Language:** TypeScript
